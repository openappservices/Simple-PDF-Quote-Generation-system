# Simple-PDF-Quote-Generation-system
Installation Instruction:
1. Clone the repository
2. From CMD or Terminal Navigar to the folder "Simple-PDF-Quote-Generation-system"
3. Make sure Docker service is up and running.
4. Run command "docker-compose up --build"
5. Site is available at http://localhost:8080/
